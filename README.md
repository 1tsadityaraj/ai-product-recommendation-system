# AI Product Recommendation System

This is a web application that helps users find products based on their needs using intelligent logic.

Users can type queries like:

laptop under 30000

The system recommends suitable products and provides links to Amazon and Flipkart.

## How the Project Works

- The user enters a product query in simple English
- The system understands the product type and budget
- Product data is collected using web scraping
- The system filters and ranks the best products
- Recommended products are shown on the website
- Users can click Amazon or Flipkart to view real products

## Why This Is an AI Project

- The system understands user intent
- It makes decisions automatically
- It gives different results for different queries
- It uses rule-based artificial intelligence

## Technologies Used

- React (Frontend)
- Node.js & Express (Backend)
- Web Scraping (Cheerio, Axios)
- Tailwind CSS
- Vercel (Deployment)

## Features

- Smart product recommendations
- Budget-based filtering
- Real-world product data
- Amazon & Flipkart redirection
- Clean and responsive UI
